## ğŸ“¦ Installation

Follow these steps to run the project locally:

1.  **Clone the repository**
    ```bash
    git clone [https://github.com/aswanayyappan/real-chat-system.git](https://github.com/aswanayyappan/real-chat-system.git)
    ```

2.  **Navigate to the project directory**

3.  **Install dependencies**
    ```bash
    npm install
    ```

4.  **Start the server**
    ```bash
    node server.js
    ```

5.  **Open the app**
    Open your browser and visit: `http://localhost:3000` (or whichever port your server runs on).
